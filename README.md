# Terminal Games

一个在终端中运行的复古游戏合集，包含俄罗斯方块和贪吃蛇两款经典游戏。

## 游戏列表

### 俄罗斯方块 (Tetris)
- 经典俄罗斯方块玩法
- 7 种不同形状的方块
- 幽灵方块预览（显示方块最终落点）
- 下一个方块预览
- 计分系统和等级系统（消除行数越多，等级越高，速度越快）

### 贪吃蛇 (Snake)
- 经典贪吃蛇玩法
- 吃食物增长身体
- 得分系统（每吃一个食物 +10 分）
- 难度递增（得分越高速度越快）

## 运行方式

```bash
go run main.go
```

## 操作说明

### 主菜单
| 按键 | 功能 |
|------|------|
| ↑ ↓ | 选择游戏 |
| Enter | 确认选择 |
| Q | 退出游戏 |

### 俄罗斯方块
| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 硬降（直接落到底） |
| P | 暂停 / 继续 |
| Esc | 返回主菜单 |

### 贪吃蛇
| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| P | 暂停 / 继续 |
| R | 重新开始 |
| Esc | 返回主菜单 |

## 项目结构

```
go-game/
├── main.go              # 主菜单入口
├── tetris/
│   ├── game.go          # 游戏逻辑
│   ├── renderer.go      # 画面渲染
│   └── main.go          # 游戏入口
└── snake/
    ├── game.go          # 游戏逻辑
    ├── renderer.go      # 画面渲染
    └── main.go          # 游戏入口
```
